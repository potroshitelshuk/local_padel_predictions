## Описание

Исследование посвящено анализу паттернов игровой активности в паделе и разработке ML-моделей для предсказания локальных исходов (исход очка, гейма, сета) на основе данных point-by-point статистики.

**Источник данных:** [Padel API](https://docs.padelapi.org/) — профессиональный API с данными Premier Padel, FIP и World Padel Tour турниров.

## Метрики

Основные метрики для оценки моделей:
- **Accuracy** — доля правильных предсказаний
- **ROC-AUC** — площадь под ROC-кривой
- **Log Loss** — логарифмическая функция потерь
- **Brier Score** — калибровка вероятностей
